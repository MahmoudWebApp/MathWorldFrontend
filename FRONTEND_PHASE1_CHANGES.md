# Frontend Phase 1 Changes

## English

- Added official and training attempt support to the problem page.
- Added solving-time, mastery, review, and attempt-history displays.
- Added the student error notebook with archive and restore actions.
- Added the error notebook shortcut to the dashboard.
- Completed the mobile navigation for guests, students, and administrators.
- Added Arabic and English translations for all new interface messages.
- Updated RTK Query types and endpoints to match the new backend APIs.

## العربية

- تمت إضافة دعم المحاولة الرسمية ومحاولات التدريب في صفحة المسألة.
- تمت إضافة عرض وقت الحل والإتقان والمراجعة وسجل المحاولات.
- تمت إضافة دفتر أخطاء الطالب مع الأرشفة والاستعادة.
- تمت إضافة اختصار دفتر الأخطاء إلى لوحة التحكم.
- تم استكمال قائمة الموبايل للزوار والطلاب والمديرين.
- تمت إضافة ترجمات عربية وإنجليزية لجميع رسائل الواجهة الجديدة.
- تم تحديث أنواع ونقاط اتصال RTK Query لتتوافق مع واجهات الباك إند الجديدة.
